# STREET BEAT

Sneakers as unique as you. E-commerce website[Demo](https://street-beat.netlify.app/).

## Tech Stack

- Frontend: React.js, HTML, SCSS
- Backend: MockApi
- Deployment: Netlify
- Other Libraries and Tools: React Router, Axios, React Context, React Skeleton.
  
## Features

- Product Listing
- Cart Management
- Wishlist Management
- Search by
  - Product Name
- Checkout
- Order Summary
- Order History
